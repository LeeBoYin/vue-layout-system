# Usage

  • Place your contents between <LayoutList> tags
<LayoutList
	padding-x="5"
	padding-y="5"
	gap="5"
	horizontal-align="center"
	style="border: 1px solid;"
>
	<button>OK</button>
	<button>Cancel</button>
	<button>Save</button>
</LayoutList>

# Props

Property Description Type Default Value
